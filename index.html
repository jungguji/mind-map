Mind Map

사용 방법

  • 노드 선택: 노드를 클릭하여 선택 (선택된 노드는 초록색, 테두리 두껍게 표시)
  • 멀티 선택: 빈 공간을 드래그하여 박스로 여러 노드를 선택
  • 노드 이동: 선택된 노드를 드래그하여 이동 (여러 노드 선택 시 함께 이동)
  • 캔버스 패닝 (데스크톱): Space 키를 누른 채로 드래그
  • 캔버스 패닝 (모바일): 2손가락으로 드래그
  • 노드 편집: 노드를 더블클릭하여 텍스트 수정
  • 자식 노드 추가: 노드를 하나만 선택한 상태에서 텍스트 입력 후 "자식 노드 추가" 버튼 클릭
  • 노드 삭제: 선택된 노드를 Delete 키로 삭제 (Root 노드는 삭제 불가, 여러 노드 동시 삭제 가능)
